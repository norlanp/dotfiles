# UI Bolder

`/ui-bolder [target]` - Amplify boring/safe designs. More visual impact, personality, memorability. Bold = distinctive, not chaotic.

## Flow

1. **Assess weakness**: generic choices, timid scale, low contrast, static, predictable, flat hierarchy
2. **Understand bounds**: brand personality, purpose, audience, constraints
3. **Plan amplification**: ONE focal point, personality direction, hierarchy amplification
4. **Amplify systematically**

## Amplification Dimensions

### Typography
- Replace system/generic fonts with distinctive choices
- Extreme scale jumps (3-5x, not 1.5x)
- Weight contrast (900 paired with 200, not 600 with 400)
- Unexpected: variable fonts, display headlines, condensed/extended widths

### Color
- Increase saturation (not neon)
- Bold palette with unexpected combinations (NOT purple-blue gradient)
- One bold color owns 60%
- Sharp high-contrast accents
- Tinted neutrals matching palette

### Spatial Drama
- Extreme scale: important 3-5x larger than surroundings
- Break the grid: elements escape containers
- Asymmetric layouts with tension
- Generous space (100-200px gaps, not 20-40px)
- Intentional overlap for depth

### Visual Effects
- Dramatic shadows for elevation (not generic drop shadows on rounded rects)
- Background: mesh patterns, noise, geometric patterns
- Texture: grain, halftone, duotone, layered — NOT glassmorphism
- Thick borders, decorative frames, custom shapes
- Custom illustrations, decorative brand details

### Motion
- Staggered dramatic page load (50-100ms delays)
- Parallax, scroll-triggered reveals
- Satisfying hover/click feedback
- ease-out-quart/quint/expo only (never bounce/elastic)

### Composition
- Hero moments with dramatic treatment
- Diagonal flows
- Full-bleed elements
- Unexpected proportions (70/30, 80/20)

## AI Slop Warning

When making things "bolder," AI defaults to cyan/purple gradients, glassmorphism, neon dark glows, gradient text. These are the OPPOSITE of bold — they're generic. Bold means distinctive, not "more effects."

**The test**: If you said "AI made this bolder," would they believe you? If yes, start over.

## NEVER

- Add effects randomly (chaos ≠ bold)
- Sacrifice readability
- Make everything bold (then nothing is)
- Ignore accessibility
- Overwhelm with motion
- Copy trendy aesthetics blindly
